# XIFAT Downloader 🎬

A modern, fast, and responsive video downloader web app built with pure HTML, CSS & JavaScript.

## ✨ Features

- **Multi-Platform Support**: Download from YouTube, Facebook, TikTok
- **Multiple Quality**: Choose from 360p, 720p, 1080p or Best Quality
- **MP3 Audio**: Extract audio only if needed
- **No Watermark**: Clean downloads for TikTok videos
- **Responsive UI**: Works perfectly on mobile & desktop
- **Dark Blue Theme**: Modern UI with smooth animations
- **No API Key**: Uses free Cobalt API - no registration needed

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: Cobalt.tools API for video processing
- **Styling**: Custom CSS with glassmorphism + neon effects
- **Hosting**: Vercel

## 📦 How to Use

1. Copy any YouTube/Facebook/TikTok video link
2. Paste it in the input field
3. Click "Get Video"
4. Select your preferred quality
5. Click "Download Now"
